Solar Analysis Process

Follow these 6 simple steps to analyze solar panel installations using our AI technology.

01

Select Location on Map

Choose location via map click or search. Coordinates auto-filled with address display.

02

Use Selected Coordinates

Transfer lat/long to input fields. Sample ID generated automatically from coordinates.

03

Initiate AI Analysis

Start analysis with 1200 sq.ft radius scan. AI processes satellite imagery automatically.

04

View 3D Satellite Results

Interactive 3D view with controls. Altitude, tilt, and imagery details displayed.

05

Review Analysis Summary

Panel detection, count, area, capacity results. Confidence score and QC status shown.

06

Download/Save Results

Download JSON report or save to database. Start new analysis or view details.
